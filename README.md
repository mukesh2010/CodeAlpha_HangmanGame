# CodeAlpha_HangmanGame

## Description
A simple text-based Hangman game built in Python as part of the CodeAlpha Python Programming Internship (Month 1, Task 1). The player guesses a hidden word one letter at a time, with a limit of 6 incorrect guesses.

## Features
- Random word selection from a predefined list
- Tracks guessed letters and remaining attempts
- Win/lose detection with clear console feedback

## Technologies Used
- Python 3
- Concepts: random, while loop, if-else, strings, lists

## How to Run
1. Clone this repository
2. Run `python hangman.py` in your terminal
3. Follow the on-screen prompts to guess letters
